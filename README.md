# FLD Leaderboard
Source code for the leaderboard used in the Fast Little Dodgers tournament

## Project setup
- Make sure you have downloaded and installed [Node.js](https://nodejs.org/en/download)
- Clone the repository into a folder of your choice
- Open a command prompt in the root of the project and run `npm install`
- Run the project locally with `npm run start`
